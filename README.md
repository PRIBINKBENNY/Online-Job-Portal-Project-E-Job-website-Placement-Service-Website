About Online Job Portal (Synopsis)
The objective of this application is to provide a system through which job seekers and recruiters can interact efficiently.

Existing System: Traditional methods rely on print and visual media, which are time-consuming and resource-intensive.
Proposed System: A web-based portal where:

Applicants register and post/update resumes.
Employers register, post job openings, and search for resumes.
Admin oversees the whole platform including user management.
Modules:
Job Seeker Module: Upload resumes, edit profiles, search jobs, receive alerts when selected.
Employer Module: Post/update job vacancies, search resumes by criteria.
Administration Module: Manage applicant/employer data and overall system integrity.
Technology Stack:
Frontend: ASP.NET, HTML, JavaScript
Backend: C#.NET
Database: MS SQL Server 2005/2008
IDE: Visual Studio .NET 2008
Hardware: Pentium IV Processor, 1GB RAM
